# linux-proxy-settings

1. Clone or download the script.
2. Add execute permissions (optional): `chmod +x enableProxy.sh`
3. Run the script: `./enableProxy.sh`
4. Root or sudo privileges are required to modify environmental variables (e.g., `/etc/environment`).
